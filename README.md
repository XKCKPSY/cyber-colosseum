# ⚔ 사이버 투기장 (Cyber Colosseum) — 실시간 MVP

실시간 키배를 벌이고, 관중이 배심원이 되어 판결하는 투기장.
**여러 사람이 진짜로 동시에 접속**해서 여론바·팝콘·판결이 실시간 동기화됩니다.

> ✅ 의존성 0개. `npm install` 필요 없음. Node만 있으면 `node server.js`로 바로 실행.

---

## 🚀 실행 방법 (1분)

1. [Node.js](https://nodejs.org) 설치 (18 이상 권장)
2. 이 폴더에서:
   ```bash
   node server.js
   ```
3. 브라우저에서 **http://localhost:3000** 접속
4. **멀티플레이 확인:** 탭을 2개 열거나, 같은 와이파이의 다른 기기에서 `http://<내PC의IP>:3000` 으로 접속 →
   한쪽에서 팝콘 던지고 투표하면 **다른 쪽 화면에 실시간으로 반영**됩니다.

---

## 🎮 써보는 순서

1. **로비**에서 불타는 링(토론) 하나 클릭
2. 싸움 보기 전 **내 입장을 비공개로 선택** (강하게A ~ 강하게B) → 입장
3. **라이브**: 상단 여론바가 실시간으로 출렁. 검투사에게 🍿·🪨 던지고, 라이브 중 언제든 입장 변경(모두의 여론바가 움직임), 채팅으로 A/B 편들기
4. **🔨 판결 내리기** → 전체 배심원 + 실명 배심원 결과 + "몇 명이 마음을 바꿨나(델타)"

## 🧠 핵심 설계 (요금·성능 안전)

이 MVP는 처음부터 **"팬아웃 폭발"을 막는 집계 패턴**으로 짰습니다. 나중에 대규모로 가도 구조를 안 갈아엎어도 되게:

- **클라이언트 스로틀** — 팝콘을 연타해도 `public/index.html`에서 0.5초 모았다가 한 번에 전송.
- **서버 집계** — `server.js`는 개별 팝콘을 저장하지 않고 방 카운터만 증가.
- **TICK 브로드캐스트** — 상태를 매 이벤트마다 쏘지 않고 `TICK_MS`(400ms)마다 방 요약 1개만 전송.
- **모두가 방 요약 1개만 구독** — 관중이 1만 명이어도 브로드캐스트 대상은 "방 상태 문서 하나".

> 이 4가지 덕분에, 재미(웨이브 연출)와 비용(팬아웃 방지)이 같은 방향으로 갑니다.

## ☁️ 배포 (실제로 남들이 접속하게)

Node 앱을 받아주는 아무 호스팅에나 올리면 됩니다. 예:

- **Render / Railway / Fly.io** — 저장소 연결 후 실행 명령 `node server.js`. `PORT`는 환경변수로 자동 주입됨(코드가 `process.env.PORT` 사용).
- **VPS(예: 저렴한 클라우드/베어메탈)** — `node server.js` + 리버스 프록시(nginx)로 443 → 3000. HTTPS면 프론트가 자동으로 `wss://`로 붙습니다.

배포 시 체크:
- [ ] HTTPS(도메인)면 WebSocket이 `wss://`로 자동 전환됨 (코드에 처리돼 있음)
- [ ] 프록시(nginx 등)에서 **WebSocket 업그레이드 헤더 허용** (`Upgrade`, `Connection`)

## ⚠️ 이 MVP의 한계 (다음 단계)

지금은 "진짜 실시간 멀티플레이가 되는 것"까지가 목표라 이렇게 단순화돼 있어요:

- **인메모리 저장** — 서버 재시작하면 상태 초기화. → 실제론 Redis(실시간 카운터)+DB(전적·판결기록)로 교체.
- **인증 없음** — 지금은 누구나 익명. → Firebase Auth 등 **전화번호 인증(1인 1표)** 붙여 부계정 어뷰징 방지.
- **검투사 = 가상** — 지금은 관중 상호작용만 실시간. → 실제 두 참가자가 발언하는 "선수 매칭"은 다음 단계.
- **단일 서버** — 동접 수천까진 이대로 OK. 수만~10만 가면 여러 서버 + 방 샤딩 + Redis pub/sub로 확장.
- **필터 없음** — 신상·욕설 자동 필터(방송 비프음식)와 판결 조작 방지 로직은 아직 미구현.

## 🧭 지켜야 할 철학 (요약)

- 판결은 **주장**에게, 사람에게 X. 신상은 누적 가명 뒤로.
- 도파민은 **역전**에서 (찍어누르기 X → 설득해야 이김).
- 패자도 잘 싸우면 **명승부상** → 선수 풀 유지.
- 실명·익명 판결은 **둘 다 표시**, 가중치 없음.
- **집계로 처리** → 재미와 비용이 같은 방향.
